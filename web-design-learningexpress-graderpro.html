Web Design: EBSCO LearningExpress

LearningExpress GraderPro www.learningexpresshub.com

Design of LearningExpress, LLC's essay grading tutorial product for teachers, GraderPro.

Tasks included:

  • Two types of interactive grading interfaces, grading for a specific topic, and grading for all topics at once
  • Display of authentic student essays with superimposed teacher grading notes in the answer view
  • Collapsible table of contents and student writing prompts, which reduce page clutter
  • Clear representation of user's answers and "GraderPro" answers, allowing for a quick comparison between user and expert grader responses
  • Optional "View Scoring Guide" modal window, which displays the full scoring criteria for each example
  • "From the Rubric": scoring criteria for each individual score, based on scoring topic and score point
  • Color coded scoring topics which help the user differentiate between many different topics
